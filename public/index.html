If it won’t load, allow inbound TCP port 3847 in this Mac’s Firewall (System Settings → Network → Firewall → Options).

Server is bound to 127.0.0.1 only. To use OBS on another computer, set HOST=0.0.0.0 in .env (or remove HOST), save, and restart npm start.

Requirements

  • Spotify app credentials in .env
  • Redirect URI must match Spotify Dashboard and .env
